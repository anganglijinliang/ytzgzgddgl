# 安钢集团永通球墨铸铁管 - 数字化转型战略规划 (2026-2027)

**版本**: 1.0  
**制定人**: 首席架构师 & 数字化专家  
**日期**: 2026-02-07

## 1. 核心愿景 (Vision)
打造行业领先的**"透明工厂" (Transparent Factory)** 与 **"智慧车间" (Smart Workshop)**，实现从"经验驱动"向"数据驱动"的转型，确保每一支球墨铸铁管的全生命周期可追溯。

## 2. 战略支柱 (Strategic Pillars)

### 2.1 一管一码，单支追溯 (One Pipe, One Code) [暂缓/Deferred]
*状态: 由于硬件设备（打码机/扫码枪）尚未就绪，此功能暂缓实施。*
目前系统主要基于**订单维度**（Order-based）管理。未来为满足高端市场（如水利工程、市政管网）的严苛要求，可下沉到**单支维度**。
- **实施方案 (待定)**: 
    - 每一支管在退火工序下线时生成唯一二维码（Unique QR ID）。
    - 标识包含：`炉号-班次-规格-流水号`。
    - 后续所有工序（水压、内衬、喷锌）扫码录入，绑定该支管的质量数据。

### 2.2 质量数字化 (Digital Quality Assurance) [当前重点/Focus]
从简单的“合格/不合格”状态记录，升级为**工艺参数记录**。无需额外硬件，只需在现有录入界面增加关键字段。
- **关键数据点**:
    - **水压试验**: 记录保压时间（秒）和压力值（MPa）。
    - **喷锌/外防**: 记录锌层厚度（g/m²）和终饰层厚度（μm）。
    - **内衬**: 记录水泥/环氧厚度。
- **价值**: 建立批次级质量档案，为客户提供更真实的MTC（材质单）数据支持。

### 2.3 生产可视化与实时看板 (Visual Management)
利用车间大屏（Digital Signage）实时展示生产进度，消除信息孤岛。
- **功能模块**:
    - **Andon系统**: 设备故障一键呼叫。
    - **实时产量**: 计划 vs 实际对比。
    - **工序平衡**: 识别瓶颈工序（如内衬固化时间过长导致堆积）。

### 2.4 移动端离线能力 (Offline-First Architecture)
考虑到铸管车间金属屏蔽严重，网络信号不稳定。
- **技术升级**:
    - 引入 **PWA (Progressive Web App)** 技术。
    - 实现生产任务的**离线缓存**：早班在有网处下载任务，车间内离线操作，下班后自动同步。
